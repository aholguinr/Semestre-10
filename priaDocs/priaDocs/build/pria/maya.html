Grúa Maya¶

Inicio de operación¶

Una vez autenticada la sesión en el computador destinado para la grúa se debe ejecutar desde la terminal el siguiente comando:

roslaunch maya_process process.launch sim_mode:=false

Una vez iniciada la interfaz gráfica de usuario se deberá enviar el robot a home haciendo clic en el botón que lleva el mismo nombre. La conexión con la base de datos remota se realiza haciendo clic en la casilla Firebase de la pestaña PRIA-UN.
